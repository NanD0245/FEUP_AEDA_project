Detailed Description

The class Movement is a class that has information about each and every movement made by vehicles between Highways, Tolls and Lanes

Parameters
typetells if it is an exit movement (leaving a Toll) if it's true or entrance (entering a Toll) if it's false
vehicleis the Vehicle object that made the movement
highwayis the Highway object in which the movement was made
tollis the Toll object that the vehicle passed by when it made the movement
laneis the Lane object that the vehicle was in when it made the movement
datewas the moment when the vehicle made the movement
