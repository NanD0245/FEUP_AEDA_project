Detailed Description

The Vehicle class is a class that gives information about the vehicle its driver is using.

Parameters
plateis a non-empty string that identifies the vehicle (e.g. 89BL32)
v_classis a number that defines the type of the vehicle there is class 1, class 2, class 3 and class 4, as such, v_class can be 1, 2, 3 or 4, according to the vehicle's class
greenlaneis true if the car driver would rather use green lanes than non green, false otherwise
taxis the value that the vehicle pays per kilometer, according to its vehicle class
