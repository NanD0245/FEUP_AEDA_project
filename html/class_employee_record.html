Detailed Description

Employees class is a class that aggregates different workers.

Each worker's information is stored in their corresponding Employee object, which is then processed in the vector 'employees' using Employees' methods.

Parameters
employeescontains all the different workers (objects from Employee class)
codeis a number that changes whenever a new worker(Employee object) is added to the vector, so each Employee has a unique code that identifies them
