Detailed Description

Highway class is a class that aggregates different tolls.

Each toll's information is stored in their corresponding Toll object, which is then processed in the vector 'tolls' using Highway's methods.

Parameters
tollscontains all the tolls (objects from 'Toll' class) that belong to this Highway
nameis a unique string (other objects can't have the same name)
