Detailed Description

The Date class is a class that provides a date that can be used and/or compared to other dates

When initialized, the user inputs a string in the format "day/month/year hour:minute:second" that is then converted to corresponding variables that will be used later on.

Parameters
s_datecorresponds to the string inputted by the user when they initialize the Date class and is in the format referred above
