Detailed Description

The class MovementOut is a class derived publicly from Movement class. It has information about a Movement that is an exit.

Parameters
entryis a Movement object that gives information about the previous movement of that vehicle (an entry movement)
distanceis the distance between the last toll (an entrance toll) and the current toll (an exit toll)
priceis the price that the vehicle driver has to pay to exit the toll, calculated using travelled distance and the vehicle's tax
